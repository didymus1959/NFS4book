# NFS4book
NFS4  Study Guide in plain German

NFSv4 (Network File System Version 4) ist eine Weiterentwicklung des Dateisystemprotokolls, das Dateien über ein Netzwerk zugänglich macht, mit Verbesserungen wie effizienterem Caching, besserer Leistung, integrierter Sicherheit (Kerberos), der Verwendung von TCP als Standardprotokoll und einem einzigen Port (2049) für Firewall-Freundlichkeit. Es führt auch einen globalen Namensraum (PseudoFS) ein, wodurch die Verwaltung von Exporten einfacher wird und es weniger Probleme mit Dateisperren gibt, insbesondere im Vergleich zu NFSv3.
